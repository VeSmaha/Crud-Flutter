# Crud em Flutter

Crud de cadastro de usuários em flutter.

## Pré-requisitos

- Flutter SDK instalado
- Dart SDK instalado

## Instalação

1. Clone este repositório em sua máquina local:
2. flutter packages get (obtem dependencias)
3. flutter run
